# Smart Study Planner

## Overview
Smart Study Planner is a Python-based application designed to help students manage academic tasks efficiently. It allows users to add, track, and organize study activities with deadlines and priorities.

This project was developed as part of the BYOP (Bring Your Own Project) capstone.

---

## Problem Statement
Students often forget assignments and struggle with time management due to lack of structured planning tools.

---

## Features
- Add study tasks
- Assign deadlines
- Set task priority
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using JSON

---

## Technologies Used
- Python
- JSON File Handling
- Git & GitHub
