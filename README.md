# Smart Study Planner

## Overview
Smart Study Planner is a Python-based application designed to help students manage academic tasks efficiently. It allows users to add, track, and organize study activities with deadlines and priorities.

This project was developed as part of the BYOP (Bring Your Own Project) capstone.

---

## Problem Statement
Students often forget assignments and struggle with time management due to lack of structured planning tools.

---

## Features
- Add study tasks
- Assign deadlines
- Set task priority
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using JSON

---

## Technologies Used
- Python
- JSON File Handling
- Git & GitHub

---


---

## How to Use
1. Run the program.
2. Choose an option from the menu:
   - Add Task
   - View Tasks
   - Complete Task
   - Delete Task
3. Enter task details when prompted.
4. Tasks are saved automatically.

---

## Working Principle
The application uses a menu-driven interface where users interact through terminal input. Tasks are stored in a JSON file, allowing data persistence between program runs.

Workflow:
1. Load tasks from file
2. Display menu options
3. Perform selected operation
4. Save updated tasks
5. Repeat until exit
